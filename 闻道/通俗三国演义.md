### 小猪佩奇列传《1》
```rust
// 兄弟姐妹之间要叫 big brother, little brother, 显得更加亲密。
 
// 猪妈妈 应该叫 Mommy Pig 英语习惯将重点放在前面.

// puddle n.小池，泥坑

// 今天下雨了 —— it is raining (today). 

// 雨停了 —— it's stopped raining. 使用现在完成时。

// If you jump in muddy puddles, you must wear your boots.
// 如果前面的句子是现实中可以发生的，后面的从句用现在进行时就行，如果是虚构的，就要用假设语气。

// 让我们再去找几个泥坑去跳吧。—— Let's find some more puddles.
// 你想要再来喝点茶吗？—— Would you like some more (tea)? 

// 玩得很开心 —— have a lot of fun.

// 强调程度 '很' 用really

// 我得确认一下……是不是……  —— I must check if ……
// 我得检查一下我带没带手机。 —— I must check if I have my phone.

// 只是……而已 —— It's only …… 
// 只是些泥而已 —— It's only mud.

// 描述刚才的某一段时间里一直在做的事，这种情况就可以用现在完成进行时
// have/has + been 加动词的 ing 形式
// I have been donging my homework.
// 你猜猜我们刚才干什么了？ —— Guess what we've been doing.

// 洗澡 —— have a bath/take a bath

// 真脏 —— look at the mess.
// 你看看你真脏 —— Look at the mess you're in.

// 快清理干净，别让妈妈看见你们这么脏。
// Let's clean up quickly before Mummy sees the mess.

// 当我们清理完之后，你和妈妈也会一起来玩吗？
// When we've cleaned up, will you and Mummy come and play, too?
// when + 现在完成时 表示 ……之后(这里的动作是现在就在做，然后说做完这个动作之后的事。)

```

### 小猪佩长列传《2》
```rust
// 吃晚饭的时候, 恐龙先生坐在乔治身旁. —— At suppertime, Mr.Dinosaur sits next to George.

// 请求他人再说一遍 —— Sorry?

// 是你吗？还是恐龙先生呢？ —— Was that you? or was that Mr.Dinosaur?

// bath n.洗澡，浴缸

// 恐龙先生和乔治睡在一起。 —— When George goes to bed, Mr.Dinosaur is tucked up with him.
// tuck sb in / tuck sb up in bed 安顿好某人，给某人盖好被子。

// 乔治最喜欢的游戏是把恐龙先生扔到空中，然后在它掉下来的时候把它接住 ——
// George's favourite game is throwing Mr.Dinosaur up in the air and catch him when he falls back down.

// 侦探也许该试着猜猜恐龙先生在哪? —— The detective could try and guess where Mr.Dinosaur might be.

// have sb do…… 让某人做……
// let sb do…… 允许某人做……
// 乔治总是在洗澡的时候让恐龙先生陪着 —— George always has Mr.Dinosaur have bath with him.
// 或 —— George always has Mr.Dinosaur with him in the bath.

// 所以它一定在这里 —— So, that's where he is.

// 我正准备说这个呢。—— I was going to say that.
// 我正准备敲门。 —— I was going to knock the door.

// 哪里都没有看到恐龙先生 —— Mr.Dinosaur isn't anywhere.

// you don't love me!  No! I do love you.  —— do 表强调

// 我在想你这一次是不是把恐龙先生扔的太高了一点呢 —— I wonder if this time you threw Mr.Dinosaur just a bit too high. 

// have …… sth back 重新找到，重新拥有，重新看到的意思都有，很抽象。
// 找回了恐龙先生，乔治开心的不得了。 —— George is so happy to have Mr.Dinosaur back again.
```
